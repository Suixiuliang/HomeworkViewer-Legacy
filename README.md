## 1. 运行环境要求

1. 系统要求

   - Windows 7 : **Windows 7 Build 7601(Windows7 SP1)** 及 **启用AERO效果**

2. .NET 环境

   本项目基于.NET 10.0开发并迁移至.NET 6.0 *(安装包默认自带)*

## 3.使用说明

- 设置：右上角的工具栏中，点击展开按钮，找到设置，在里面调整功能（包括自定义、调整字号、等等）

- 检查更新：可根据Github镜像站检查更新。如果你更新失败（安装包下不下来），请先检查自己的网络问题，如果您的网络连Github主站也没有问题，那么请提交Issues
- 编辑作业：点击编辑按钮，即可编辑作业
- 轮播模式：工具栏中的轮播按钮
- 历史记录等其它功能请自己探索......
